# JobsNP Github Repo
JobsNP is an online job portal web application.

## Tech Stack
- Django Web Framework
- Postgresql
- HTML
- Bootstrap CSS
- FontAwesome Icons


### Setup venv and database then Run
```sh
pip install -r requirement.txt
py manage.py makemigrations
py manage.py migrate
py manage.py runserver
```
